Class Clock

source code

object --+    
         |    
 Singleton --+
             |
            Clock

A Simple clock class to allow for retrival of time from multipule threads to be more efficient. This class is a singleton so anyone using it will be using the same instance.

The clock updates every 100ms so calls to get time often can be more performant if they dont need to be exact.
