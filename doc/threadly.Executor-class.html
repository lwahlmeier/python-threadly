schedule(self, task, delay=0, recurring=False, key=None)

source code 

Schedule a task for execution.

Parameters:
  • delay (number) - amount of time to delay the task from running.
  • recurring (boolean) - set to True if the task is to we rerun. We will rerun it after every delay time
  • key (Object) - This sets a key to the task. Any task with this key will be executed in a single threaded manor.

removeScheduled(self, task)

source code 

Remove a scheduled task from the queue. This is a best effort remove, the task could still possibly run. This is most useful to cancel recurring tasks. If there is more then one task of this type scheduled only the first one is removed.

Parameters:
  • task (function pointer) - task to remove from the queue

shutdown(self)

source code 

Shutsdown the threadpool. Any task currently being executed will still complete, but nothing in the queue will run.
